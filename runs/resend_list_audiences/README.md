# Resend List Audiences

Retrieves a list of all audiences from Resend.

## Environment Variables

| Name | Type | Description |
|------|------|-------------|
| `resend_api_key` | text | API key for Resend |

## Inputs

None required.

## Response

Returns an array of audiences with their IDs and details.
