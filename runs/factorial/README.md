# Factorial

Calculates the factorial of a given number (n!).

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `number` | int | Yes | The number to calculate factorial for |

## Response

Returns the factorial of the input number.

## Example

**Input:**
```json
{
  "number": 5
}
```

**Output:**
```json
120
```

## Notes

The factorial of n (written as n!) is the product of all positive integers less than or equal to n. For example, 5! = 5 × 4 × 3 × 2 × 1 = 120.
