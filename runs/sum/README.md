# Sum

Calculates the total sum of an array of numbers.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `numbers` | decimal[] | Yes | Array of numbers to sum |

## Response

Returns the sum of all numbers in the array.

## Example

**Input:**
```json
{
  "numbers": [1, 2, 3, 4, 5]
}
```

**Output:**
```json
15
```
