# Reverse Array

Reverses the order of elements in an array.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `array` | json | Yes | The array to reverse (also works with strings) |

## Response

Returns a new array with elements in reverse order.

## Example

**Input:**
```json
{
  "array": [1, 2, 3, 4, 5]
}
```

**Output:**
```json
[5, 4, 3, 2, 1]
```
