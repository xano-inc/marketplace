# Concat Arrays

Concatenates multiple arrays into a single array.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `array_1` | json[] | Yes | First array |
| `array_2` | json[] | No | Second array |
| `array_3` | json[] | No | Third array |
| `array_4` | json[] | No | Fourth array |
| `array_5` | json[] | No | Fifth array |
| `array_6` | json[] | No | Sixth array |
| `array_7` | json[] | No | Seventh array |
| `array_8` | json[] | No | Eighth array |
| `array_9` | json[] | No | Ninth array |

## Response

Returns a single array containing all elements from the provided arrays in order.
