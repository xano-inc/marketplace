# Sum of Squares

Calculates the sum of squares of an array of numbers.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `numbers` | decimal[] | Yes | Array of numbers |

## Response

Returns the sum of each number squared: x₁² + x₂² + x₃² + ...

## Example

**Input:**
```json
{
  "numbers": [1, 2, 3, 4]
}
```

**Output:**
```json
30
```

(1² + 2² + 3² + 4² = 1 + 4 + 9 + 16 = 30)
