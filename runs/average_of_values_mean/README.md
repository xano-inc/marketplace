# Average of Values (Mean)

Calculates the arithmetic mean (average) of an array of numeric values.

## Inputs

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `values` | decimal[] | Yes | Array of numbers to calculate the average from |

## Response

Returns the arithmetic mean of the provided values as a decimal number.

## Example

**Input:**
```json
{
  "values": [1, 2, 3, 4, 5]
}
```

**Output:**
```json
3
```
