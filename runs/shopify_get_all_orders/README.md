# Shopify Get All Orders

Retrieves a list of orders from Shopify using the GraphQL Admin API.

## Environment Variables

| Name | Type | Description |
|------|------|-------------|
| `access_token` | text | Shopify Admin API access token |
| `store` | text | Shopify store domain |

## Inputs

None required.

## Response

Returns a list of orders with pagination information.
